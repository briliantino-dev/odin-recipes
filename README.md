# odin-recipes

This is my first official web development project from [The Odin Project Foundations](https://www.theodinproject.com/) course. It is a simple recipe website built using HTML.

## Description
The goal of this project is to build a basic recipe site containing a main index page with links to a few recipe pages. The styling is kept minimal (plain HTML) as the primary focus is on mastering document structure and link navigation before moving on to CSS.

## Skills Demonstrated
By completing this project, I have successfully demonstrated the following skills:
* **HTML Foundations:** Structuring documents, using appropriate headings, paragraphs, and list elements (ordered and unordered).
* **Links and Images:** Implementing relative anchor paths to navigate between local pages and embedding images using proper attributes.
* **Git & GitHub Workflow:** Initializing, cloning, committing changes, and pushing them to a remote repository from the command line.
* **File Management:** Organizing project files and directories cleanly on a Linux environment.
